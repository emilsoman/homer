# Homer


Homer makes tracking your Unix dotfiles easay peasay

-Currently under development and not yet available -

## Installation

    gem install homer
## Usage

- Initialize homer :

        homer init
- Start tracking dotfiles :

        homer add ~/.vimrc
- List tracked files :

        homer list
- Push tracked files to GitHub

        homer push
